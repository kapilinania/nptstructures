## 🚀 Features

- 📱 **Responsive Design**: Flawlessly adapts to desktop, tablet, and mobile displays.
- 🌓 **Dark/Light Mode**: Integrated theme toggling for an optimal user experience.
- 🧊 **Interactive 3D Models**: Embedded video and interactive 3D drawing showcases.
- 🔍 **SEO Optimized**: Fully optimized meta tags, schema markup, and performance tuning for high search engine rankings.
- 🎨 **Modern UI/UX**: Features clean layouts, fluid animations, and a professional aesthetic.

## 💻 Local Development

To run this project locally for development or testing:

<details>
  <summary><b>Show Setup Instructions</b></summary>
  
  <br>

  1. **Clone the repository:**
     ```bash
     git clone https://github.com/yourusername/nptstructures.git
     ```
  2. **Navigate to the project directory:**
     ```bash
     cd nptstructures
     ```
  3. **Run the site:**
     Since this is a static website, you can simply open `index.html` in your web browser. For the best experience, we recommend using a local development server like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code.
</details>
